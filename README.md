# AustriaODL.github.io

## 奧地利 ODL 華人團的 Wiki 網站

這是一個為奧地利遠距工作/數位遊牧華人團體建立的文檔中心網站。

## 📖 項目介紹

本項目是一個靜態 Wiki 網站，用於整理和分享奧地利 ODL (Online/Distance Learning) 華人團的相關資訊、資源和文檔。

## 🌐 線上訪問

訪問 [https://austriaodl.github.io](https://austriaodl.github.io) 查看完整網站內容。

## ✨ 功能特點

- **響應式設計**：支援桌面和移動設備瀏覽
- **字體優化**：使用 JetBrains Mono 字體，自動 fallback 到本地字體
- **繁體中文介面**：為華人社群提供友好的閱讀體驗
- **靜態網站**：無需後端伺服器，直接透過 GitHub Pages 部署

## 🚀 本地開發

### 預覽網站

由於這是純靜態網站，您可以直接在瀏覽器中打開 `index.html` 文件進行預覽：

```bash
# 方法 1: 直接使用瀏覽器打開
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows

# 方法 2: 使用簡單的 HTTP 伺服器
python3 -m http.server 8000
# 然後訪問 http://localhost:8000
```
